# 369 The Dev Store

Simple one-page landing page for a web design company.

## Stack

- Next.js
- Supabase for enquiry storage
- Nodemailer for email notifications

## Local setup

1. Install dependencies.
2. Copy `.env.example` to `.env.local` and fill in the values.
3. Create a Supabase table named `enquiries`.
4. Run the app with `npm run dev`.

## Supabase table

Use a table like this:

```sql
create table public.enquiries (
  id bigint generated by default as identity primary key,
  name text not null,
  email text not null,
  company text,
  budget text not null,
  message text not null,
  created_at timestamptz not null default now()
);
```

## Notes

- The page is intentionally kept to one screen and uses only red, black, and white.
- The logo asset is in `public/logo.svg`.
